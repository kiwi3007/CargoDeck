# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Open a [GitHub Security Advisory](https://github.com/kiwi3007/CargoDeck/security/advisories/new) instead. You should receive a response within 48 hours.

Please include:
- Type of issue (e.g. SQL injection, XSS, auth bypass)
- File paths and location of affected code
- Steps to reproduce
- Proof-of-concept if possible
- Potential impact

> Note: CargoDeck is intended for use on a trusted local network only. It is not recommended to expose it to the public internet.
